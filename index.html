Hockey Showdown

This was my first app that I made which got me into CS. This is a object oriented multiplayer hockey game created using Java, OOP and SWING based GUI on Windows.






Overview

  • Made to make a portable, quick hockey game
  • 3 minute, 1 on 1 hockey game
  • Game Modes include Single Player, Multi-Player and Battle for the Cup
  • Choose between the Toronto Maple Leafs, Montreal Canadiens, New York Rangers and Boston Bruins


Cool Features

Developed A.I. algorithm for computer as opposing team with interactive GUI and multimedia



Zombieland

This was my first app that I made in a (3-person) team using Git Source control development. This is a Object oriented Zombie game developed using Java, OOP and LibGDX (Windows).






Overview

  • Made to be like Call of Duty Zombies
  • Zombies come in waves where each wave has more zombies that spawn at an exponential rate
  • Player has option of buying new guns throughout the game with acquired points to kill zombies more easily
  • New guns increase speed of shot and bullets in round


Cool Features

Developed A.I. algorithms for computer based opponent to find fastest route through game maze to break down barricade and find the player
